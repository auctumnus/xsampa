# xsampa
Converts [X-SAMPA](https://en.wikipedia.org/wiki/X-SAMPA) to 
[IPA](https://en.wikipedia.org/wiki/International_Phonetic_Alphabet). 
Includes types and hopefully fully correct conversion.

## Installation

```sh
$ yarn add xsampa
or
$ npm install xsampa
```
